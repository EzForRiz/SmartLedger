# 💰 SmartLedger

SmartLedger is a Django-based personal finance management system that helps users track expenses, analyze spending habits, and gain financial insights.

---

## 🚀 Features

- User authentication system
- Add / edit / delete expenses
- Category-wise tracking
- Dashboard with analytics
- Interactive charts (planned)
- AI-based spending insights (future scope)

---

## 🛠 Tech Stack

- Python
- Django
- SQLite
- HTML, CSS, JavaScript
- Chart.js (planned)
